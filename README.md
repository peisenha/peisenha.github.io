# peisenha.github.io

*Personal website for Philipp Eisenhauer — science, engineering, and decision-making at scale*

Organizations invest in science but struggle to turn evidence into decisions fast enough.
The gap between knowing and doing is where value gets lost — in manual handoffs, ad hoc
analysis, and judgment calls made without systematic feedback.

This site documents how Philipp bridges that gap: building production systems that
translate scientific models into actionable decision frameworks, and sharing the operating
principles behind the work.

Visit the live site at [peisenha.github.io](https://peisenha.github.io).
